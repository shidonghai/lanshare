1. About Zapya

1. What is Zapya ?

Zapya is a cross-platform mobile service which allows users to discover and share digital content. It is the fastest and most convenient mobile sharing tool.

2. What are the core functionalities of Zapya？

Zapya assists mobile users to achieve fast and massive file transfer. Using Zapya, the user can share digital content on the phone with friends including music, video, photo and files of any other formats.

3. Is Zapya a Bluetooth-based service？

No, Zapya is not using Bluetooth technology for file transfer. Zapya surpasses Bluetooth in effective transfer speed which can reach 50 times of Bluetooth speed at the maximum gap.

Zapya can support transfer at longer distance up to about 50 meters, and allow multiple people to connect and share simultaneously.

4. Can Zapya be used without WiFi network？Does Zapya use mobile data for file transfer ?

Zapya does not require nor depend on WiFi-based Wireless Local Area Network (WLAN). Zapya users can share files without mobile data service nor SIM card.

Zapya uses direct wireless connection between devices and thus incurs no mobile data traffic during file transfer.

5. How do I acquire Zapya Prestige Credits and Points ?

The user must register and login into Zapya, and send SMS to invite a new user to successfully install Zapya in order to acquire 1 Prestige Credit and the matching Points.

Participating in Zapya Points activities could exchange for various gifts. The activity, prize and rules will differ in each session.

  1. Connect to Friends

1. How to help nearby friends to install Zapya for first time ?

1）SMS Invite: Click 'Invite' button, select intended recipients from Contacts list or enter phone number directly to send SMS. The recipient can click on Zapya download link from received SMS to download and install Zapya installation app, which then retrieves the full app from the sender's device or from the web site.

2 ) No-mobile-data Installation: Select "Invite with zero mobile traffic" option, the user needs to create a new connection to let friends nearby to acquire Zapya directly. There will be no need of external network nor incur mobile traffic. After creating a new connection, the user should guide friends to follow operation instructions in order to acquire Zapya from a given web page or from scanning a QR code.

3）Direct Download: If connected in a WiFi network or having plenty mobile data quota, the user can download from Zapya web site or scan QR Code for quicker download. After installation, friends can start connecting to friends for file sharing.

2. Is Zapya limited to one-to-one connection？

No, one-to-one connection is the limitation of Bluetooth. Zapya is created to enhance the user experienced by allowing up to 5 people participating in the same group.

3. How do Zapya users establish connections？

Clicking "Connect to Friends" button to enter the screen to select options of connecting to friends.

While being part of existing WiFi network, the user can click the network or user name shown in the list of available Zapya users to create a new group connection or to click the "Join" button to join an existing group. The user can also click the "Create Group" button to select group members to be invited and thus creates own WiFi group connection.

When not in existing WiFi network, the user can join an already created group connection or create a new HotSpot group connection.

When there is no existing WiFi group connection nor WiFi HotSpot group connection, then one person must select to Create Group under HotSpot tab while the other person must search for the new group connection and then click to join.

4. How does Zapya user create secure connection？

For ease of use and less intrusive obstacle, Zapya defaults to not creating secure connection. The user can create secure connection under "HotSpot Connection". The user must go to "Settings -> Privacy" screen to select "Set connection password" option. Once set, the new members joining the group connection must know the password. If setting a password after creating a HotSpot group connection, then the new password would be effective at next time creating a group connection.

5. What is the functionality of White List？

The user may find the White List option in Zapya settings screen. The recorded values are Zapya friends who can join your network without your approval each time. If the user wants to remove the privilege to certain friends, the user may delete the matching records in the White List so that those friends would require approval to join the user's connection next time.

6. While I am already in a group, can I invite new friends to join ?

Yes, the Zapya user can invite new friends to join Zapya group at any time.

For user who created the group, it is simply verbally inviting a new friend to join the group that carries the group owner's nickname.

For user who joined in one group already, using Zapya to invite a new friend means leaving the connected group and creating a new group to wait for the new friend.

7. How can I request one member of a group to leave ？

Click on the avatar image, select the option to leave group. This privilege is only reserved to the user who created the group.

8. Why I sometimes can no longer connect to the Internet while using HotSpot group connection？

To reduce usage of mobile data traffic, Zapya sets default behavior of making HotSpot connection and 3G/4G mobile connection as mutually exclusive. When starting a HotSpot group, the previously enabled or disabled 3G/4G mobile network would become disabled and thus no Internet connection. When joining a HotSpot group, the mobile OS would route data traffic to WiFi interface which connects to the HotSpot device that has 3G/4G mobile network disabled and thus no Internet connection. Once the HotSpot group is finished, the previous settings would all be restored.

9. How do I get online while in WiFi HotSpot connection ?

Zapya also provides users the setting to allow "HotSpot Connection" co-exist with "Mobile Network". To enable the feature, the user must start from "Settings -> Network" to find and select "Share mobile data network" option. This setting is only valid for the user who creates a group connection. Moreover, once this feature is enabled, all mobile devices joining the group connection would be able to share the mobile data network; therefore, please enable this feature with discretion.

  1. File Transfer

1. What types of files can Zapya transfer?

Zapya intends to impose no limits on the types of files to transfer and to ease file selection process. One exception: Zapya iOS version does not allow sharing of iOS Apps due to restrictions of Apple App Store.

2. How do I send a file？

There are 4 ways to send a file:

1）Press and hold file icon, wait until icon is enlarged, the flick the phone

2）Press and hold file icon, wait until icon is enlarged, drag and drop to recipient avatar

3）Click file icon, select "Share" option from the popped up menu

4）Use one finger to press and hold recipient avatar, wait until avatar is enlarged, then use other finger to repeated click the files to be shared.

3. During file transfer, can the recipient reject the transfer?

Yes. In the Settings screen, the user can select "Receive Confirmation" option. If selected, the user will see an extra option next to action dialog in History screen which allows the recipient to Allow or Reject.

4. If losing connection during file transfer, can file transfer being resumed from the break point？

File transfer will stop immediately when detecting connection being lost. When the connection is re-established, the transfer operation that was paused will be automatically resumed from the break point.

  1. Other Functionalities

1. What is the "History" screen in Zapya?

The tab at the bottom-right of the main screen shows Zapya "History". In this screen, the user can find all the interaction records with friends. From each record, the user can play music or video, and view photos.

2. How to search for more exciting content？

The "Hot" content page of Zapya integrates some of the most intriguing mobile games, the most interesting apps, the most exciting video and the most popular music. You and your friends can not only transfer and share the content on individual mobile devices, you can also acquire more exciting new content from the resources of "Hot" screen.

3. How do I recommend Zapya to my friends on Weibo？

Clicking the "Search Friends' option of the app menu would allow you to send invitation to friends not registered yet in Zapya. This options supports friends on Tencent Weibo and Renren network.

  1. Interactive Games

1. There are interactive network games on Zapya platform？

Yes. Zapya provides interesting "Ice Hockey", "Doodle Toss' and "Sango Jump" games. Friends in a group connection can enjoy the fun of playing network games among mobile devices.

2. How to start network games with friends？

Zapya network games must be installed separately after installing Zapya. The user can select to "Download" and "Install" from the game tab or to retrieve from a friend in the same group connection who already has installed the game. After installing or upgrading the new game, the friends in a group connection can start the network game.

3. What are the game rules of Ice Hockey？

When two players start the game, use hand to control the blue paddle to move freely for offense and defense.

The border line at the bottom of the screen is owner's goal line, the red hockey ball or puck crossing the goal line means the opposite player scores.

The border line at the top of the screen is the middle field line, the red hockey ball or puck crossing that line goes over to the opposite player's half of the field. The opposite player would respond by moving paddle to defend and attack.

The player scores one point at a time and the first player reaches 7 points would win the game. After finishing the game, both players can choose to continue the ice hockey game.

4. How do I play the Doodle Toss game？

After connecting to friends in a group connection, click "Doodle Toss" game icon and then select "Start" to enter the game. The user can import photos or just doodle freely and then share the end work with friends.

5. What are the game rules of "Sango Jump" game？

Every player select one role avatar to start the game. Whoever jumps to the finish line first would win the game in 2-player option. There would be various game props being randomly injected into the game. Acquiring the game props could help to gain speed or to cause troubles to the opposite player.
